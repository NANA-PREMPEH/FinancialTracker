How Exchange Rates Work

  • • Exchange rates are used when you add expenses in a different currency than your wallet's currency.
  • • The system automatically fetches rates from an online API if no manual rate is set.
  • • Manually set rates take priority and won't be overwritten by auto-fetch.
  • • Rates are cached for 24 hours to reduce API calls.
